### Auto Twitch PlateUp! Visit

This userscript creates button for sending "!visit" to chat when streamer playing PlateUp!

#### üß© How it works
Once you start watching stream, it check current playing game and if its PlateUp!, creates a button to send "!visit" to chat.
if button clicked, it will focus the input area, type "!visit" and send.
While focusing if chat rules appear, it wait until accepted, but if closed before acceptence it can't do nothing.
You can accept chat rules automatically but its against the Twitch rules. Use it with responsibility.

#### ‚öôÔ∏è Main features
- Write "!visit" with one-click.
- Detect chat rules and wait until acceptence.
- Colorfull and understandable logging in console.

#### üîí Privacy and safety
- The script does **not** collect or send any user data.
- It contains **no tracking, ads, or hidden requests**.
- All operations run locally in your browser on the Twitch website.

#### üí° Notes
- You can adjust the some variables in the code to change how code works.
- Not work when using popup chat while checking game, if you use popup chat change `CHECK_GAME` variable to `false`.
